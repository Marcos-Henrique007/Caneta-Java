# Caneta Java

Projeto desenvolvido em Java com o objetivo de simular/representar o funcionamento de uma caneta, utilizando conceitos de Programação Orientada a Objetos (POO).

---

## Sobre o Projeto

O **Caneta Java** é um projeto simples e didático, ideal para quem está aprendendo Java.  
Ele demonstra conceitos fundamentais como:

- Classes e Objetos
- Atributos e Métodos
- Encapsulamento
- Modificadores de acesso
- Instanciação de objetos

---

## Funcionalidades

- Criar objetos do tipo `Caneta`
- Definir atributos como:
  - Cor
  - Marca
  - Ponta
  - Carga (nível de tinta)
  - Tampa (aberta/fechada)
- Métodos como:
  - Tampar / Destampar
  - Escrever
  - Verificar estado da caneta

---

## Conceitos aplicados

- Programação Orientada a Objetos (POO)

- Boas práticas em Java

## Como executar

1. Clone o repositório:
git clone https://github.com/Marcos-Henrique007/Caneta-Java.git

2. Abra na sua IDE Java

3. Execute a classe principal (`Main.java`)

---

## Exemplo de uso

```java
Caneta c1 = new Caneta();

c1.cor = "Azul";
c1.ponta = 0.5f;

c1.destampar();
c1.escrever();
```
